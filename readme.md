# vorane
Declarative image editor.

## Build instructions
Prerequisites:
- C++20 compatible compiler
- GLFW (e.g. mingw-w64-x86_64-glfw on MSYS2)
- Desire to create
```sh
$ make # build the project, provide -jN to use N parallel jobs
$ ./bin/vorane.exe # run the built executable

# or all in one step
$ make run
```
